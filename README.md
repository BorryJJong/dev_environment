### docker compose

```
├── docker-compose.yml  # docker container 명세
├── mysql               # mysql 관련 폴더
│   └── my.cnf          # mysql 환경 설정 파일
```

### cli

```
docker-compose ps -a

docker-compose up
docker-compose up -d
docker-compose down

docker-compose config
docker-compose version

```
